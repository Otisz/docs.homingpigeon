# Charge schema
A charge representing the fee for a service.

## Used in
- [Daily closing - `otherCharges`](/schema-ntak/daily-closing#othercharges)
- [Daily closing - `postStayCharges`](/schema-ntak/daily-closing#poststaycharges)
- [Checkout-day transaction](/schema-ntak/segments/checkout-day-transaction#charges)

## Properties
### `amount`
- Type: `number`
- Rules: `required`

<Callout type="info">The amount must be converted to Hungarian Forint.</Callout>

### `category`
- Type: `enum`
- Rules: `required` | [`oneOf: Charge item category (enum)`](/schema-ntak/enums/charge-item-category)

### `isTouristTax`
- Type: `boolean` | `null`
- Rules: `required`

<Callout type="info">This value can be null if it's in `dailyClosing.otherCharges`, because it will be ignored anyway.</Callout>

### `taxPercentage`
The percentage value of VAT payable on the product or service.
- Type: `number`
- Rules: `required`

## Object
```ts
export type Charge = {
  amount: number;

  isTouristTax: boolean | null;

};
```
