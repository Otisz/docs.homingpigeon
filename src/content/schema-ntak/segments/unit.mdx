## Properties
### `building`
Name of the building.
- Type: `string` | `number` | `null` | `undefined`

<Callout type="info">If not present or `null`, the fallback value will be the letter "**a**".</Callout>

### `number`
Name of the room, plot, etc...
- Type: `string` | `number`
- Rules: `required`

### `type`
Type of the room, plot, etc...
- Type: `enum`
- Rules: `required` | [`oneOf: Unit type (enum)`](/schema-ntak/enums/unit-type)

### `singleBeds`
Number of single beds in the room.
- Type: `number`
- Rules: `required` | `min: 0`

### `doubleBeds`
Number of double beds in the room.
- Type: `number`
- Rules: `required` | `min: 0`

### `extraBeds`
Number of double beds in the room.
- Type: `number` | `null` | `undefined`
- Rules: `min: 0`

<Callout type="info">If not present or zero, this value won't be included in the report.</Callout>

## Object
```ts
export type Unit = {

  singleBeds: number;

};
```
