# Checkout guest schema

## Used in
- [Checkout](/schemas/viza/checkout#guests)

## Properties
### `guestId`
A unique identifier indicating the arrival of a specific guest.
The same value must be submitted for the guest's arrival and departure to ensure that the arrival and departure can be linked.
- Type: `string`
- Rules: `required` | `uuid` | `must be the same as what you used at checkin`

### `departedAt`
- Type: `string`
- Rules: `required` | `format: 'Y-m-d H:i:s' UTC`

export type CheckoutGuest = {
